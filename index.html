Un programme de formation aux bases du développement web Simplon-co / EMG

découvrir et aborder les technologies et les outils du développement web 'Front End'

  • HTML
  • CSS
  • JavaScript

Les ressources mises en oeuvre :

  • openclassrooms.com
  • freecodecamp.org
  • ...
  • un bon brin de curiosité et d'astuce

l'exercice

Après une introduction succincte aux fonctionnalités asynchrones de JavaScript (AJAX), une page doit présenter aléatoirement des citations en utilisant une api fournissant un fichier .json

Les étapes à suivre

  • la recherche d'une ressource
  • le choix d'une présentation
  • la mise en forme (html/css)
  • le code (js)
  • les tests
  • la présentation sur codepen.io
